# README

Este projeto inclui:

- **app/**: front-end Vue 3 (Vite)  
- **server/**: backend Express (Node.js)

## Pré-requisitos

- Node.js instalado  
- npm disponível

## Passos

1. **Clone o repositório**  
  ```bash
  git clone <URL>
  cd <pasta-do-projeto>

2. **Instale e rode o servidor**  
  cd server
  npm install
  npm start

3. **Instale e rode o app**  
  cd ../app
  npm install
  npm run dev
